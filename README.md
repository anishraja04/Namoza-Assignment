# Task 1 – GTM Event Schema

## Objective

The goal of this task is to design a complete Google Tag Manager (GTM) event tracking plan for OrthoNow's website. The tracking plan covers important user interactions such as appointment booking, call button clicks, WhatsApp clicks, patient guide downloads, clinic page views, and blog engagement.

### Deliverables

- Complete GTM Event Schema
- Booking Funnel Tracking
- dataLayer JSON examples
- GTM Trigger Mapping
- GA4 Funnel Explanation
- Google Ads Conversion Recommendation
